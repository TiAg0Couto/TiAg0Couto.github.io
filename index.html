Glossário

HTML (HyperText Markup Language)
Linguagem de marcação utilizada para estruturar o conteúdo na web.
CSS (Cascading Style Sheets)
Linguagem de estilo utilizada para definir a aparência e o layout de páginas web.
JavaScript
Linguagem de programação utilizada para criar interatividade e funcionalidades dinâmicas em websites.
React
Biblioteca JavaScript para construir interfaces de utilizador reutilizáveis e eficientes.
Angular
Framework JavaScript para desenvolvimento de aplicações web de página única (SPA).
SQL (Structured Query Language)
Linguagem utilizada para gerir e manipular bases de dados relacionais.
NoSQL
Categoria de sistemas de gestão de bases de dados que não utilizam o modelo relacional tradicional, permitindo maior flexibilidade e escalabilidade.
Segurança Cibernética
Práticas e tecnologias utilizadas para proteger sistemas informáticos, redes e dados contra ameaças e ataques.
